# MovieList 重構專案

## 專案目的

練習使用 React 框架搭配 MUI 和 TypeScript 語法進行開發。

## 功能特色

- **Theme**: 可切換深色和淺色主題。
- **Page**: 可切換 Movie 和 Favorite 畫面。
- **Mode**: 可切換 Card 和 List 模式。
- **SearchBar**: 輸入關鍵字搜尋電影。
- **Poster**: 顯示基本海報資訊。
- **More**: 點擊 More 按鈕可以顯示更多資訊。
- **Favorite/Delete Icon**: 點擊 Favorite/Delete Icon 可以新增/移除收藏電影。
- **Pagination**: 點擊分頁碼 可切換顯示分頁。
